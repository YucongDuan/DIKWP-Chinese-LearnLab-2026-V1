# DIKWP Chinese LearnLab 2026 V1

A standalone, offline-first reference prototype for English-background learners of Chinese/Mandarin.

## Workflow
Learner profile -> HSK/YCT/ACTFL/CEFR/AP bridge -> M0-M5 route -> pinyin/tone clinic -> hanzi component lab -> vocabulary chunks -> grammar patterns -> input/output tasks -> Evidence Ledger -> Error Ledger -> AI Use Log -> teacher/peer review -> Chinese Learning Passport.

## Quick start
Open `index.html` in a modern browser. No server, database, API key, account, model endpoint, or network is required.

Optional CLI:
```bash
python tools/run_chinese_plan.py examples/sample_learner.json --out outputs
```

## Boundaries
This system does not issue HSK, YCT, ACTFL, CEFR or AP results; it does not automatically place, grade or certify learners; it does not conduct hidden audio recording or biometric pronunciation scoring.
